## Md5

```php
$faker->md5(); // 368f9d4f93fa6ac519bb6c951fd44ef8
```

## Sha1

```php
$faker->sha1(); // 2882fe0eb3bc73fcbef51572d0403efa1c48bb08
```

## sha256

```php
$faker->sha256(); // 8d7edc63ab003719669e96b83d76c89c86dede8dbfa04d7c37ea42e9fd2e28f9
```
