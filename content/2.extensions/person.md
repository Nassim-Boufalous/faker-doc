---
title: Person
description: Person extension for Xefi Faker PHP
---

## Name
Gender: `M` for male, `F` for female and `null` for both
```php
$faker->name(gender: null); // Jane Doe
```

## First Name
Gender: `M` for male, `F` for female and `null` for both
```php
$faker->firstName(gender: null); // John
```

## Last Name
```php
$faker->lastName(); // Doe
```

## Title
Gender: `M` for male, `F` for female and `null` for both
```php
$faker->title(gender: null); // Prof.
```
