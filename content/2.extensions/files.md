---

## Requirement

This extension is not included with the base faker php package, you'll need to require it:

```bash
composer require --dev xefi/faker-php-files
```

## Mime Type

```php
$faker->mimeType(); // application/pdf
```

## File Extension

```php
$faker->fileExtension(); // pdf
```
